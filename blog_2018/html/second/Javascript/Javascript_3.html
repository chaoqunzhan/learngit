1、有关数组的题目，slice()、filter()、concat()、reverse(),哪个操作会改变原数组？

  • slice():返回截取的新数组，slice（start,end）中的两个参数分别是start：起始位置和end：结束位置。只传递一个参数，默认为其实位置参数，截取到数组末尾。传递的参数有负数，就加上数组长度确认，如：数组长度4，slice(-3,-1)=slice(1,3)
  • filter():返回筛选数组，对数组的每一项都运行，返回结果为ture的元素组成心数组。 
        var words=[12,3,4,33,56];
    var words_filter=words.filter(function(word){
        return word>10;
    });       //运行结果：words_filter=[12,33,56]
  • concat():返回合并数组，就是合并两个数组
  • reserve():反转数组的方向，操作原数组，返回新数组 
        var words=[1,2,3,4,5,6];
    var words_re=words.reserve();       
    //运行结果：words_filter=[12,33,56]

有关数组的知识点可以参考前端早读课

2、关于移动端事件touchstart、touchmove、touchend

  • touchstart:当手指触摸屏幕时候触发，即使已经有一个手指放在屏幕上也会触发。
  • touchmove:当手指在屏幕上滑动的时候连续地触发。在这个事件发生期间，调用preventDefault()事件可以阻止滚动。
  • touchend:当手指从屏幕上离开的时候触发。

3、关于canvas中的drawImage()的三种不同的使用方法：

  • drawImage(img,x,y)，img指定图像，x和y指定图像在画布Canvas上的坐标。
  • drawImage(img,x,y,width,height)，width和height是指定图像的大小，通过指定该值可以将图像进行缩放和放大
  • drawImage(img,sx,sy,sw,sh,x,y,w,h)，使用该方法可以在图像上进行裁剪，并将裁剪的区域显示在指定的位置。其中sx和sy指定裁剪区域的起始坐标，该坐标是相对于图片的，不是画布。sw和sh是裁剪区域的大小。

4、http状态码，这实在是记不住，太多了，常见的有：

  • 200：请求成功
  • 301：资源（网页等）被永久转移到其它URL
  • 404：请求的资源（网页等）不存在
  • 500：内部服务器错误

5、关于HTML中localStorage储存问题：

在html5中的Web Storage包括了两种存储方式：sessionStorage和localStorage。sessionStorage不是一种持久化的本地存储，只是会话级别的存储。而localStorage用于持久化的本地存储，除非主动删除数据，否则数据是永远不会过期的。
  • localStorage.setItem("name","value"):存储变量名为name，值为value的变量
  • localStorage.getItem("name"):获取存储的变量name的值
  • localStorage.removeItem("name"):删除变量名为name的存储变量

7、js中undefined与null的问题，其实还有NaN：

  • undefined：表示未定义类型，就是当声明的标量未初始化时，该标量的默认值就是undefined。
  • null：表示空类型，是一个“什么都没有”的占位符，值undefined实际上是null派生出来的，所以在js中它们是相等的。 
      alert(null==undefined);      //返回的值是ture
    尽管两个值相等，但是含义不一样，undefined是声明了变量但没有赋值，而null是给变量赋了一个空值。
  • NaN：是number中的一个比较特殊的值，表示非数（Not a Number）。 
    typeof(NaN);    //返回的值是number

8、jsnop的工作原理：

JSONP是一种非正式传输协议，该协议的一个要点就是允许用户传递一个callback或者开始就定义一个回调方法，参数给服务端，然后服务端返回数据时会将这个callback参数作为函数名来包裹住JSON数据，这样客户端就可以随意定制自己的函数来自动处理返回数据了。JSONP和JSON的区别：JSON是一种传输格式，而JSONP呢是一种数据的获取方式。JSONP可以跨域，记住这一点就可以了。

9、js实现继承的方式：

js中的继承跟java不一样，在java中有类的概念，而js中没有。所以在js中的继承一般是通过对象和对象来实现的，具体的方法可以参考js中实现继承的六种方法。

10、简单说明map、foreach、filter、every、for的遍历方式：

  • map:map()方法会迭代数组中的每一个元素，并根据回调函数来处理每一个元素，最后返回一个新数组。但是，这个方法不会改变原始数组。 
        var ary = [1,2,3,4];
    var res = ary.map(function (item,index,input) {
        return item*10;
    })
    console.log(res);   //运行结果[10,20,30,40];
    console.log(ary);   //运行结果[1,2,3,4]；
  • foreach:foreach()和map()一样也是遍历数组中每一个元素，不同的是foreach是没有返回值的,而且会对原数组产生影响。 
        var ary = [1,2,3,4];
    var res = ary.foreach(function (item,index,input) {
        return item*10;
    })
    console.log(res);   //运行结果undefined;
    console.log(ary);   //运行结果[10,20,30,40]；
