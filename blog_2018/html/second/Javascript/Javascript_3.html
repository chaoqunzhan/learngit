原本还以为，前端的试题应该会是HTML+CSS+js的试题，想不到美图的笔试题竟然几乎全是js，有的题目实在是没见过，下面就做个简单的总结，学习学习。体型记得是：选择题（10*4分）+简答题（3*10分）+编程题（2*15分），好像有点出入，忘记了，大概吧！

选择题：

1、有关数组的题目，slice()、filter()、concat()、reverse(),哪个操作会改变原数组？

  • slice():返回截取的新数组，slice（start,end）中的两个参数分别是start：起始位置和end：结束位置。只传递一个参数，默认为其实位置参数，截取到数组末尾。传递的参数有负数，就加上数组长度确认，如：数组长度4，slice(-3,-1)=slice(1,3)
  • filter():返回筛选数组，对数组的每一项都运行，返回结果为ture的元素组成心数组。 
        var words=[12,3,4,33,56];
    var words_filter=words.filter(function(word){
        return word>10;
    });       //运行结果：words_filter=[12,33,56]
  • concat():返回合并数组，就是合并两个数组
  • reserve():反转数组的方向，操作原数组，返回新数组 
        var words=[1,2,3,4,5,6];
    var words_re=words.reserve();       
    //运行结果：words_filter=[12,33,56]

有关数组的知识点可以参考前端早读课

2、关于移动端事件touchstart、touchmove、touchend

  • touchstart:当手指触摸屏幕时候触发，即使已经有一个手指放在屏幕上也会触发。
  • touchmove:当手指在屏幕上滑动的时候连续地触发。在这个事件发生期间，调用preventDefault()事件可以阻止滚动。
  • touchend:当手指从屏幕上离开的时候触发。

3、关于canvas中的drawImage()的三种不同的使用方法：

  • drawImage(img,x,y)，img指定图像，x和y指定图像在画布Canvas上的坐标。
  • drawImage(img,x,y,width,height)，width和height是指定图像的大小，通过指定该值可以将图像进行缩放和放大
  • drawImage(img,sx,sy,swidth,sheight,x,y,width,height)，使用该方法可以在图像上进行裁剪，并将裁剪的区域显示在指定的位置。其中sx和sy指定裁剪区域的起始坐标，该坐标是相对于图片的，不是画布。swidth和sheight是裁剪区域的大小。

4、http状态码，这实在是记不住，太多了，常见的有：

  • 200：请求成功
  • 301：资源（网页等）被永久转移到其它URL
  • 404：请求的资源（网页等）不存在
  • 500：内部服务器错误
