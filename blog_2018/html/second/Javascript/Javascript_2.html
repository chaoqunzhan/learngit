jQuery图片自动轮播效果

时间:2018-03-16  作者:小提群

总结一下整个轮播效果的实现过程，以下分为几个部分：

用到的jQuery函数，css样式等：

1、setInterval("调用的函数名",周期性调用函数之间间隔的毫秒数)

2、animate的使用方法：$(selector).animate({params},speed,callback);

  • params:必选，定义形成动画的css属性
  • speed:可选，规定效果的时长速度
  • callback:可选，是动画完成后执行的函数名

3、css中的position属性，其中包括：

  • absolute:生成绝对定位的元素，相对于static定位以外的第一个父元素进行定位。元素的位置通过 "left", "top", "right" 以及 "bottom" 属性进行规定。
  • relative:生成相对定位的元素，相对于其正常位置进行定位。因此，"left:20" 会向元素的 LEFT 位置添加 20 像素。
  • fixed:生成绝对定位的元素，相对于浏览器窗口进行定位。元素的位置通过 "left", "top", "right" 以及 "bottom" 属性进行规定。
  • static:默认值。没有定位，元素出现在正常的流中（忽略 top, bottom, left, right 或者 z-index 声明）。
  • inherit:规定应该从父元素继承 position 属性的值。

4、css中的样式选择器nth-child、first-child、last-child

  • first-child表示选择列表中的第一个标签
  • last-child表示选择列表中的最后一个标签
  • nth-child(n)表示选择列表中的第n个标签
