object、embed、video、audio标签的区别

在HTML5中的object、embed、video、audio标签都可以添加视频或音频文件,但是它们之间还是有着细微的区别，如下：

object和embed用来添加视频或音频的时候，在页面加载后是自动播放的，不能进行手动控制播放；对于它们的height和width属性，是可以任意更改的，不受视频的比例影响。

video和audio中,可以通过改变autoplay的属性来控制播放，具有比较好的用户体验；播放控制条是自带的,而且一直显示在播放的固定位置；在调整容器的大小方面,video标签会相应的进行等比缩放,你更改其中的width或者height另一个也会进行变动，对于audio标签,只有width属性,更改height时,展示形式为调整它相对于容器的高度。

 

用video标签实现视频的外链插入网页

在做网站的时候，有时候要插入视频，这个时候有两个问题：一个是像我们这种个人的网站或者博客租用的虚拟空间有限，如果把视频也上传到虚拟空间上，那么空间很快会被占满；另一个问题就是视频加载速度的问题，如果所有的信息都来自于同一台主机，那么速度可想而知，肯定相当的不理想。
